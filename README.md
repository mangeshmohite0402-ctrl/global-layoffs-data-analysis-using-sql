# global-layoffs-data-analysis-using-sql

Cleaned and analyzed a real-world dataset of 2,361 companies affected by layoffs (2020–2023)

Built an end-to-end SQL data pipeline for data cleaning, transformation, and analysis

Removed duplicates using CTEs and window functions

Standardized inconsistent data (dates, industries, country names)

Handled missing values using self-joins and data imputation techniques

Performed exploratory data analysis (EDA) to identify industry, country, and time-based trends

Developed rolling monthly trend analysis to track layoff patterns over 3 years

Identified key insights:

Highest layoffs: Amazon (18,150)

Peak year: 2023 (125,677 layoffs)

Most impacted sectors: Consumer industry & Post-IPO companies

Total layoffs analyzed: 383,159
